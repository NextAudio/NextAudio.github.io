    Requirements

    NextAudio targets .NET 6.

    We recommend to use NextAudio in windows or any linux distribution, but we don't recognize any other OS limitations for now.

    NextAudio currently don't support ARM processors, if you wanna this support for your app please discuss in our repository.

    If you need some audio glossary concepts you can see then in glossary.
